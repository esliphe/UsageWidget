# UsageWidget

UsageWidget 是一个面向 Windows 桌面的本地时间记录与使用分析小组件。它可以自动记录程序运行、前台注视、网页浏览、视频播放、音乐播放和学习内容，并通过轻量化悬浮小窗、数据详情页、图表和报表帮助用户理解自己的电脑使用习惯。

## 项目定位

这个项目不是简单的进程计时器，而是一个偏个人效率与数字生活分析的桌面工具。它会把“程序是否运行”“用户当前是否正在看某个窗口”“网页是否在前台”“浏览器是否在播放视频”“后台音乐是否正在播放”等信息拆开记录，避免把所有时间粗暴地混成一个数字。

典型使用场景：

- 了解每天真正注视和操作电脑的时间。
- 区分前台工作、后台运行、视频播放、音乐播放和学习时间。
- 统计 B 站、YouTube、课程网站等平台上的学习视频时长。
- 分析常用程序、常访问网站、常看视频和常听音乐。
- 查看一天中不同小时段的使用节奏。
- 通过学习目标、分类汇总和报告导出复盘自己的时间安排。

## 核心功能

### 进程与前台注视统计

- 自动采样 Windows 正在运行的 `.exe` 程序。
- 统计每个程序的总运行时长、前台注视时长和后台运行时长。
- 支持忽略系统关键进程和用户自定义忽略列表。
- 支持键鼠空闲检测，长时间无输入时可停止累计前台注视时间。
- 支持暂停记录，适合临时不想纳入统计的场景。

### 网页与窗口标题记录

- 可记录当前前台窗口标题。
- 对 Chrome、Edge、Firefox、Brave、Opera、Vivaldi、Arc 等浏览器窗口进行网页注视统计。
- 配合浏览器扩展后，可获取当前标签页 URL、域名、网页标题和正在发声的标签页。
- 未安装扩展时，也可回退到窗口标题级统计。
- 支持隐私模式：网页仅保留域名，普通窗口标题脱敏为程序名。

### 视频与音乐播放统计

- 通过 Windows 系统媒体会话识别正在播放的媒体内容。
- 浏览器来源的媒体会话会优先归入视频播放，避免网页视频误记为音乐。
- 酷狗音乐、网易云音乐、Spotify 等独立播放器会归入音乐播放。
- 浏览器扩展可上报正在发声的网页标签页，用于统计网页视频、课程视频和后台播放时长。
- 支持音乐标题解析与音乐分析，按歌曲和歌手聚合播放时长。

### 学习内容识别

- 内置学习主题识别，支持编程、算法、数据结构、计算机网络、操作系统、数据库、机器学习、人工智能、物理、化学、生物、高等数学、线性代数、英语、论文科研等主题。
- 支持中英文混合标题匹配，例如课程、教程、公开课、讲解、期末、备考、刷题、大学物理、光学等关键词。
- B 站、YouTube 等平台上的学习视频会自动归入“学习”分类，同时保留视频播放时长。
- 针对中文学习视频标题做了增强，例如：

```text
【完整版】《大学物理-光学》1.5小时快速突击|期末不挂科必备 观看全免费 期末首选【蜂考】
```

该类标题会识别为：

```text
分类：学习
学习主题：物理
```

### 分类与联网增强

- 内置常见程序、网站和内容分类规则，包括学习、编程、AI 工具、聊天、社交、音乐、视频、游戏、娱乐、办公、工具、购物、新闻、系统软件等。
- 支持按应用名、域名、网页标题关键词添加自定义分类规则。
- 支持右键纠正分类，纠正后会写入规则并影响后续统计。
- 可选联网分类增强，默认关闭。
- 中文联网查询优先使用 Baidu 摘要，再回退 DuckDuckGo / Wikipedia，更适合中国大陆网络环境。
- 隐私模式下会自动禁用联网分类，避免标题、域名或程序名发送到外部服务。

### 数据可视化与报表

- 概览页展示今日前台、学习、视频、音乐等关键指标。
- 分类占比环形图展示不同活动类别的时间分布。
- 24 小时时间分布图按小时展示前台、视频、音乐和学习节奏。
- 日历热力图展示近 84 天的学习、前台、视频和音乐分布。
- 本周与上周对比图展示近期变化趋势。
- Top 程序、Top 网站、Top 视频、Top 音乐条形图帮助快速定位主要时间去向。
- 数据详情页支持程序、网页、视频、音乐、窗口标题、学习分析、分类汇总、目标和时间线等视图。
- 支持导出 CSV 和 HTML 报告。

### 目标与复盘

- 支持学习目标、娱乐限制、视频限制、编程时长等目标管理。
- 目标可按分类、域名、程序或学习主题设置。
- 支持达标、接近达标、未达标状态展示。
- 支持连续达标追踪和每日自动摘要。
- 数据详情页可按今天、昨天、最近 7 天、最近 30 天等范围查看。

## 界面特点

- 无边框、置顶、可拖动的半透明小组件。
- 支持展开视图和紧凑折叠视图。
- 系统托盘常驻，关闭窗口后不会直接退出程序。
- 小组件显示今日前台、学习、视频、音乐等即时摘要。
- 状态中心可查看当前前台程序、当前网页、当前视频、当前音乐、浏览器扩展状态、媒体会话状态和数据库路径。
- 运行诊断页可查看采样耗时、数据库大小、扩展连接状态、最近联网分类来源和错误信息。

## 数据存储与隐私

UsageWidget 的数据默认保存在本机 SQLite 数据库中：

```text
%LOCALAPPDATA%\UsageWidget\usage.db
```

隐私相关设计：

- 所有核心统计均可在本地完成。
- 联网分类和联网音乐校验默认关闭。
- 开启隐私模式后，会自动禁用联网增强功能。
- 隐私模式下网页只显示域名，窗口标题会脱敏。
- 支持暂停记录。
- 支持数据库备份和数据清理。

## 性能策略

- 默认采样间隔约为 1.5 秒，在实时性和性能之间做平衡。
- 采样写入使用 SQLite 批量事务，减少频繁磁盘写入。
- 设置、忽略列表和分类规则使用内存缓存。
- 详情页聚合查询做限频刷新，避免大量数据下 UI 卡顿。
- 小组件隐藏到托盘后，会跳过不必要的可视化刷新。
- 时间线详细事件支持保留策略，避免长期使用后数据库无限增长。

## 运行环境

- Windows 10 / Windows 11
- Python 3.10 或更高版本
- 主要依赖：PySide6、psutil、pywin32
- 数据库：SQLite

## 从源码运行

在项目目录执行：

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python run.py
```

启动后会出现桌面小组件。关闭窗口时程序会隐藏到系统托盘，真正退出需要右键托盘图标选择退出。

## 打包为 EXE

项目包含 PyInstaller 配置，可使用：

```powershell
.\.venv\Scripts\python.exe -m PyInstaller -y UsageWidget.spec
```

构建结果位于：

```text
dist\UsageWidget\UsageWidget.exe
```

## 浏览器扩展

项目内置 `browser-extension` 目录，用于 Chrome / Edge 等 Chromium 内核浏览器。

安装方式：

1. 打开浏览器扩展管理页。
2. 开启开发者模式。
3. 选择“加载已解压的扩展程序”。
4. 选择项目中的 `browser-extension` 文件夹。
5. 保持 UsageWidget 桌面程序正在运行。

扩展会把当前活动标签页和正在发声的标签页发送给本机桌面程序，用于更准确地统计网页注视和网页视频播放。

```text
├── run.py                         # 程序入口
├── requirements.txt               # Python 依赖

└── usage_widget
    ├── main.py                    # 应用启动
    ├── ui.py                      # PySide6 界面
    ├── monitor.py                 # 采样与监控逻辑
    ├── storage.py                 # SQLite 数据层
    ├── learning.py                # 学习主题识别
    ├── online_category.py         # 联网分类增强
    ├── media.py                   # 媒体会话读取
    ├── browser_bridge.py          # 浏览器扩展本地桥接
    ├── content.py                 # 内容记录与分类辅助
    ├── diagnostics.py             # 诊断日志
    └── startup.py                 # 开机自启动
```

- 喜欢使用轻量桌面小组件查看状态的人。

## 注意事项

- 本工具用于个人时间统计和效率复盘，不建议用于未经同意的他人监控。
- 浏览器 URL / 标题统计、窗口标题统计、媒体标题统计都属于较敏感数据，请根据需要开启隐私模式。
- 不同播放器是否能被识别，取决于它是否向 Windows 系统媒体会话暴露标题和播放状态。
- 联网增强依赖外部搜索服务，可用性会受网络环境影响；本地规则仍是主要分类路径。
