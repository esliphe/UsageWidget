# UsageWidget v5.45.8-stable

## 修复与优化

- 打包随附 Noto Sans SC 字体，并在 PyInstaller spec 中强制校验字体文件，降低中文显示为方框的风险。
- 24 小时时间分布图从栈叠柱改为分组柱，避免前台、视频、音乐三类非互斥数据被误读为可相加总量。
- 学习标记改为柱上方圆点，X 轴在宽窗口下增加 3 小时间隔标注，并补充 Y 轴参考刻度和峰值标签。
- BarChart 统一柱色，不再按排名改变亮度，避免用户把颜色深浅误解为数据重要性。
- 进程行进度条明确标注“前台占比”，tooltip 同步解释前台时长、运行时长和占比口径。
- 热力图改为与全局色板一致的蓝色系，增加月份列标签、峰值摘要、峰值格描边和颜色图例。
- 分类图超过 5 类时自动使用横向条形图；环形图中心文本明确为“注视+播放”。
- 所有自绘图表的关键尺寸改为按 DPI 缩放，并为空状态增加虚线占位图形。
- 统计页顶部控制区增加面板分组和分隔线，当前状态页增加上次刷新时间。
- 暗色主题增强边框与弱化文字对比度；悬浮窗状态指示增加形状/符号辅助，减少对红黄绿纯颜色编码的依赖。
- 悬浮窗标题栏按钮改为符号按钮并保留 tooltip，降低固定宽度中文按钮的理解和布局风险。

## 验证

- `python -m compileall -q usage_widget run.py UsageWidget.spec` 通过。
- Qt offscreen 渲染烟测覆盖 BarChart、HourlyActivityChart、HeatmapWidget 和 DonutChartWidget 通过。
- 不写 `.pyc` 的源码语法检查通过。
