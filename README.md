# UsageWidget

> Windows 桌面时间记录与使用分析小组件 — 自动追踪程序、网页、视频、音乐和学习，帮你理解时间都去了哪里。

---

## 它能做什么

UsageWidget 是一个轻量的 Windows 桌面小组件，在后台持续记录你的电脑使用行为，并通过悬浮窗、图表和报表帮你复盘时间。与简单的计时器不同，它会区分**前台注视**和**后台运行**，分辨**视频播放**和**音乐播放**，并自动识别**学习内容**——不会把所有时间粗暴地混成一个数字。

**一句话：装上就忘掉，想看时打开看。**

---

## 核心能力

###  追踪维度

| 维度 | 记录内容 | 数据来源 |
|------|---------|---------|
| **程序使用** | 每个 `.exe` 的前台注视时长、后台运行时长 | 系统进程采样 |
| **网页浏览** | 网页标题、域名、URL、前台停留时长 | 浏览器扩展 + 窗口标题 |
| **视频播放** | B 站/YouTube/课程视频的播放时长与内容分类 | 浏览器扩展 + 媒体会话 |
| **音乐播放** | 歌曲名、歌手、播放时长 | Windows 媒体会话 + 浏览器有声标签 |
| **学习内容** | 学习主题（编程/数学/物理/英语等 40+ 类） | 本地规则 + 可选联网增强 |
| **窗口活动** | 任意前台窗口的标题与停留时长 | 系统窗口检测 |

###  智能分类

内置 **500+ 条分类规则**，可将所有活动自动归入：

`学习` `编程` `AI 工具` `聊天` `音乐` `视频` `游戏` `娱乐` `办公` `工具` `购物` `新闻` `系统`

分类来源透明可查：
- **本地规则** — 内置默认规则 + 用户自定义
- **联网分类** — 可选，通过搜索引擎摘要自动判断未知内容
- **用户纠正** — 一键纠错，写入规则永久生效，且不会被联网结果覆盖

###  学习主题识别

自动识别 40+ 种学习主题，覆盖：

- **计算机**：编程、算法、数据结构、计算机网络、操作系统、数据库、机器学习、AI、嵌入式、EDA/PCB、MATLAB、CAD
- **数学**：高等数学、线性代数、概率统计、离散数学
- **理科**：物理、化学、生物
- **人文**：英语、日语、历史、政治、经济学、法学、医学、设计
- **科研**：论文、学术研究

对中文学习内容做了专门优化——B 站上"大学物理-光学""期末不挂科""蜂考"这类视频会自动识别为 `学习 → 物理`，而不是笼统地标成"视频"。

###  可视化分析

| 图表 | 说明 |
|------|------|
| **24 小时时间分布图** | 按小时展示前台、视频、音乐、学习节奏，悬浮可看详情 |
| **分类占比环形图** | 各类活动的时间占比一目了然 |
| **日历热力图** | 近 84 天的学习/前台/视频/音乐强度分布 |
| **本周对比图** | 本周 vs 上周各时段变化趋势 |
| **Top 排行榜** | 最常用程序、最常访问网站、最常看视频、最常听音乐 |
| **周期对比** | 任意周期与上一等长周期的指标对比 |

###  目标管理

- 自定义目标：学习时长、娱乐上限、视频限制、编程时长等
- 目标可绑定分类、域名、程序或学习主题
- 每日自动摘要，连续达标追踪
- 达标/接近/未达标状态一目了然

###  数据导出

- **CSV 导出**：程序使用、网页浏览、视频播放、音乐播放、学习分析
- **HTML 报告**：完整的时间分析报告，跟随当前日期范围

### ️ 隐私设计

- 所有数据存储在 **本地 SQLite 数据库**，不上传任何服务器
- 联网分类和联网音乐校验 **默认关闭**
- **隐私模式**：网页仅保留域名，窗口标题脱敏为程序名，自动禁用联网功能
- 支持 **暂停记录**，随时停止数据采集
- 支持数据库备份与清理

## 界面一览

**桌面悬浮窗** — 无边框、可拖拽、半透明，始终置顶：

- **折叠模式**：紧凑显示今日前台时间、学习时间、视频时间、音乐时间和当前播放曲目
- **展开模式**：完整进程列表、分类统计、快速操作按钮
- 鼠标悬停自动展开，移走自动折叠
- 关闭窗口 → 隐藏到系统托盘，真正退出需右键托盘图标

**数据统计** — 多页签详情窗口：

- 概览 · 趋势 · 音乐 · 学习 · 目标 · 时间线 · 数据明细
- 日期范围支持：今天 / 昨天 / 最近 7 天 / 最近 30 天 / 本周 / 上周 / 自定义
- 上一段 / 下一段 / 今天 快捷切换

**设置中心**：

- 通用设置 · 隐私与功能 · 时间管理 · 忽略进程 · 分类规则 · 目标管理
- 分类规则表支持增删改查，显示来源与匹配方式
- 空闲阈值与时间线保留策略可精确调节

## 快速开始

### 环境要求

- Windows 10 / Windows 11
- Python 3.10+
- SQLite

### 从源码运行

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python run.py
```

启动后桌面出现悬浮小组件。关闭窗口会隐藏到系统托盘，真正退出请右键托盘图标选择"退出"。

### 打包为 EXE

```powershell
.\.venv\Scripts\python.exe -m PyInstaller -y UsageWidget.spec
```

构建结果：`dist\UsageWidget\UsageWidget.exe`

### 安装浏览器扩展（推荐）

扩展用于获取网页标题、URL 和有声标签，提升网页和视频统计精度。

1. 打开 Chrome / Edge 扩展管理页面
2. 开启"开发者模式"
3. 选择"加载已解压的扩展程序"
4. 选择项目中的 `browser-extension` 文件夹
5. 保持 UsageWidget 桌面程序在运行

## 项目结构

```text
.
├── run.py                         # 程序入口
├── requirements.txt               # Python 依赖
├── UsageWidget.spec               # PyInstaller 打包配置
├── browser-extension/             # 浏览器扩展（Chrome/Edge）
│   ├── manifest.json
│   ├── service_worker.js          # 后台：上报活动标签页与有声标签
│   └── content_script.js          # 注入页面：提取标题与媒体状态
└── usage_widget/
    ├── main.py                    # 应用启动入口
    ├── ui.py                      # PySide6 完整界面（主窗口 + 所有对话框 + 图表组件）
    ├── monitor.py                 # 核心采样引擎（进程/窗口/媒体/浏览器数据采集与分类）
    ├── storage.py                 # SQLite 数据层（建表/迁移/CRUD/聚合/导出/规则管理）
    ├── learning.py                # 学习主题识别（40+ 主题本地匹配 + 联网查询）
    ├── online_category.py         # 联网内容分类（多搜索引擎摘要 + 置信度评分）
    ├── media.py                   # Windows 媒体会话读取（WinRT API）
    ├── music_lookup.py            # 联网音乐校验（iTunes/MusicBrainz/Last.fm）
    ├── browser_bridge.py          # 浏览器扩展本地桥接 HTTP 服务
    ├── content.py                 # 窗口/网页内容类型判断
    ├── classification.py          # 标题清洗与规范化
    ├── diagnostics.py             # 诊断日志（带轮转）
    ├── startup.py                 # Windows 开机自启管理
    ├── idle.py                    # Win32 空闲检测
    ├── native_window.py           # Win32 前台窗口检测
    └── timefmt.py                 # 时间格式化工具
```

## 数据存储

数据库位置：

```text
%LOCALAPPDATA%\UsageWidget\usage.db
```

核心表结构：

| 表 | 用途 |
|---|------|
| `usage_daily` | 每日每进程的前台/运行时长 |
| `content_usage_daily` | 每日每内容项（网页/视频/音乐/窗口）的注视/后台时长、分类、学习主题 |
| `category_rules` | 分类规则（本地默认 / 用户纠正 / 联网学习） |
| `goals` + `goal_streaks` | 目标定义与每日达标记录 |
| `timeline_events` | 事件级活动时间线 |
| `settings` | 所有应用配置 |

## 注意事项

- 本工具用于**个人时间统计与效率复盘**，不建议用于未经同意的他人监控
- 浏览器 URL/标题、窗口标题、媒体标题属于敏感数据，请按需开启隐私模式
- 音乐播放器的识别取决于其是否向 Windows 系统媒体会话暴露播放信息
- 联网增强依赖外部搜索引擎，可用性受网络环境影响；本地规则仍是主要分类路径
