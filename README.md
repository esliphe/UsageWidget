# UsageWidget

UsageWidget 是一款 Windows 本地时间记录与使用分析桌面小组件，支持进程运行、前台注视、网页浏览、视频播放、音乐播放和学习内容统计，并提供分类汇总、学习主题识别、24 小时视图、热力图、目标复盘、CSV/HTML 报告导出和隐私模式。

## 功能亮点

- 记录每个 `.exe` 程序的运行时长、前台注视时长和后台运行时长。
- 记录浏览器网页标题、域名和 URL，支持 Chrome / Edge 扩展桥接。
- 通过 Windows 媒体会话和浏览器扩展统计网页视频、课程视频、后台音乐和独立播放器内容。
- 自动识别学习内容和学习主题，例如编程、算法、物理、高等数学、英语、论文科研等。
- 中文学习视频标题增强识别，例如“大学物理-光学”“期末不挂科”“蜂考”等内容会归入学习时间。
- 支持学习、编程、AI 工具、聊天、音乐、视频、游戏、娱乐、办公、工具、购物、新闻等分类。
- 支持 24 小时时间分布图、分类占比图、日历热力图、本周对比图和 Top 排行图。
- 支持目标管理、连续达标追踪、每日摘要、CSV 导出和 HTML 报告。
- 支持隐私模式、暂停记录、本地 SQLite 存储、数据库备份和数据清理。

## 典型场景

- 复盘每天真正注视和操作电脑的时间。
- 区分工作、学习、娱乐、视频和音乐时间。
- 统计 B 站、YouTube、课程网站上的学习视频时长。
- 分析常用程序、常访问网站、常看视频和常听音乐。
- 查看一天中不同小时段的使用节奏。
- 通过目标和报表复盘时间安排。

## 数据与隐私

数据默认保存在本机 SQLite 数据库中：

```text
%LOCALAPPDATA%\UsageWidget\usage.db
```

核心统计均可在本地完成。联网分类和联网音乐校验默认关闭；开启隐私模式后会自动禁用联网增强，网页只显示域名，普通窗口标题会脱敏为程序名。

## 运行环境

- Windows 10 / Windows 11
- Python 3.10 或更高版本
- PySide6
- psutil
- pywin32
- SQLite

## 从源码运行

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python run.py
```

启动后会出现桌面小组件。关闭窗口时程序会隐藏到系统托盘，真正退出需要右键托盘图标选择退出。

## 打包 EXE

```powershell
.\.venv\Scripts\python.exe -m PyInstaller -y UsageWidget.spec
```

构建结果位于：

```text
dist\UsageWidget\UsageWidget.exe
```

## 浏览器扩展

项目内置 `browser-extension` 目录，用于 Chrome / Edge 等 Chromium 内核浏览器。

安装方式：

1. 打开浏览器扩展管理页。
2. 开启开发者模式。
3. 选择“加载已解压的扩展程序”。
4. 选择项目中的 `browser-extension` 文件夹。
5. 保持 UsageWidget 桌面程序正在运行。

扩展会把当前活动标签页和正在发声的标签页发送给本机桌面程序，用于更准确地统计网页注视和网页视频播放。

## 说明

本工具用于个人时间统计和效率复盘，不建议用于未经同意的他人监控。浏览器 URL / 标题统计、窗口标题统计、媒体标题统计都属于较敏感数据，请根据需要开启隐私模式。
